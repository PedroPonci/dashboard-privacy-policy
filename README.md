Política de Privacidade
Última atualização: 29 de janeiro de 2026

1. Introdução
Esta Política de Privacidade descreve como a extensão Dashboard Viewer - MAR Advocacia ("nós", "nosso" ou "extensão") coleta, usa e protege as informações dos usuários.

2. Informações que Coletamos
Nossa extensão coleta apenas as informações necessárias para seu funcionamento:

Credenciais de Login: CPF e senha fornecidos voluntariamente para autenticação
Token de Sessão: Gerado após login bem-sucedido para manter a sessão ativa
Dados de Acesso: Informações sobre dashboards acessados e timestamp de última atualização
Importante: Não coletamos dados de navegação, histórico, informações pessoais adicionais ou qualquer outro dado além do necessário para o funcionamento da extensão.

3. Como Usamos as Informações
As informações coletadas são utilizadas exclusivamente para:

Autenticar o usuário no sistema
Manter a sessão do usuário ativa
Buscar e exibir os dashboards autorizados
Verificar permissões de acesso baseadas em grupos
4. Armazenamento de Dados
Os dados são armazenados de duas formas:

Armazenamento Local: O token de sessão é armazenado localmente no navegador do usuário usando a API Chrome Storage. Esses dados permanecem no dispositivo do usuário e não são transmitidos para terceiros.
Servidor: Credenciais (senha criptografada com bcrypt) e informações de dashboards são armazenadas em nosso servidor seguro hospedado no Railway.
5. Compartilhamento de Informações
Não compartilhamos, vendemos ou alugamos informações pessoais dos usuários com terceiros.

As únicas situações em que dados podem ser acessados por terceiros são:

Provedor de hospedagem (Railway) - apenas para operação técnica da infraestrutura
Quando exigido por lei ou ordem judicial
6. Segurança
Implementamos medidas de segurança para proteger suas informações:

Senhas criptografadas usando bcrypt (hash seguro)
Comunicação via HTTPS (conexão criptografada)
Tokens de sessão com expiração automática
Controle de acesso baseado em permissões de grupos
7. Permissões da Extensão
A extensão solicita as seguintes permissões:

storage: Para armazenar o token de sessão localmente no navegador
host permissions: Para conectar ao servidor de dashboards (https://dashboards-mar-advocacia-production.up.railway.app)
Não solicitamos permissões desnecessárias.

8. Código Remoto
A extensão não utiliza código remoto. Todo o código JavaScript está incluído no pacote da extensão e pode ser auditado.

9. Retenção de Dados
Mantemos suas informações enquanto sua conta estiver ativa. Você pode solicitar a exclusão de seus dados a qualquer momento entrando em contato conosco.

10. Direitos do Usuário
Você tem o direito de:

Acessar seus dados pessoais
Solicitar correção de dados incorretos
Solicitar exclusão de sua conta e dados
Revogar permissões desinstalando a extensão
11. Cookies e Rastreamento
A extensão não utiliza cookies de terceiros nem ferramentas de rastreamento analítico. Não monitoramos seu comportamento de navegação.

12. Menores de Idade
Nossa extensão não é direcionada a menores de 18 anos. Não coletamos intencionalmente informações de menores.

13. Alterações nesta Política
Podemos atualizar esta Política de Privacidade periodicamente. Alterações serão publicadas nesta página com a data de atualização revisada.

14. Lei Aplicável
Esta política está em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018) do Brasil.
